# ITMO Megabattle Taplink

Статичная мобильная страница для GitHub Pages.

## Публикация

1. Залейте репозиторий на GitHub.
2. Откройте `Settings -> Pages`.
3. В `Build and deployment` выберите `Deploy from a branch`.
4. Source: ветка `main`, папка `/docs`.
5. Когда купите домен, укажите его в `Settings -> Pages -> Custom domain`.

Если домен уже известен, можно создать в этой папке файл `CNAME` с доменом без `https://`.
